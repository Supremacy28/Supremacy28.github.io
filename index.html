Add Bot to Server

Sofi Bot Status

  • /setnotifychannel – Channel where Sofi Status will post
  • /setpingrole [role] – Pings a specific role about the Sofi Status

Suggestion Commands

  • /setsuggestionchannel – Sets current suggestion channel
  • /suggest [Message] – Turns it into a nice Embed

Sofi Bot commands

  • /sofiguide – Guide for Sofi bot

Bug Reporting

  • /setbugchannel – Sets current Bug channel
  • /setbugcommentrole [Role] – Allows Developer role to comment to Bug Reports
  • /reportbug – Allows users to report bugs in nice embeds
  • Update Button – Allows users to update/change their bug Reports
  • Comment Button – Allows Set Bug Role to comment to users Bug Reports
